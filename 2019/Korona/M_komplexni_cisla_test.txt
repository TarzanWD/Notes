Urči reálnou a imaginární část čísla z.

z = (1+2i / 1-2i)^2 - (1-2i / 1+2i)^2

Lze vyřešit substitucí.

A = 1+2i
B = 1-2i

z = (A / B)^2 - (B / A)^2
z = (A/B + B/A)(A/B - B/A)
z = ((AA + BB) / AB)((AA - BB) / AB)
z = (AA + BB)(AA - BB) \ (AB ^ 2)
z = (A^4 - B^4) / (AB ^ 2)

Spočteme si hodnoty členů, které budeme dosazovat

A^4 = (1 + 2i)^4 = 1 + 2^4
B^4 = (1 - 2i)^4 = 1 - 2^4
AB = (1 + 2i)(1 - 2i) = 5
AB^2 = 25

z = (1 + 2^4 - 1 + 2^4) / 25
z = 2^8 / 5^2
